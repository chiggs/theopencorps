Overview

The Open Corps is an online platform providing a cloud-based toolflow for FPGA and ASIC projects.

We leverage a selection of services that have revolutionised software development and apply them to hardware development. Combined with the best commercial tools around we have created a powerfull online flow.

You can:

  • Run simulation regressions automatically
  • Build documentation automatically
  • Build supporting software libraries
  • Run synthesis automatically
  • Easily leverage other Open Source projects as dependencies
